# AWS Master Roadmap

> AWSの全サービスを体系的に学ぶためのロードマップ

## 学習方針

各サービスは次の流れで学習します。

- サービスの目的を理解する
- コンソールで触る
- AWS CLIで操作する
- SDKで操作する
- Terraformで構築する
- AWS CDKで構築する
- 他サービスと連携する
- 料金体系を確認する
- 制約・ベストプラクティスを整理する
- 完了チェックを付ける
